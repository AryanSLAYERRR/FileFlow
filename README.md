<h1 align="center">FileFlow ‚Äî Fast, Simple File Sorter</h1>
<p align="center">
Organize your downloads in seconds. Preview every move, resolve conflicts safely, and undo in one click ‚Äî on Windows, macOS, and Linux.
</p>

---

## ‚ú® Highlights
- üîé Live Preview: stream of planned actions (MOVE ‚Ä¢ SKIP ‚Ä¢ CONFLICT) before touching files  
- üóÇÔ∏è Smart Organization: sort by extension into folders like Images, Docs, Videos, Audio, Archives, Code  
- üß© Custom Rules: map extensions (e.g., .pdf, .jpg) to destination folders in the built‚Äëin Rules Editor  
- üõ°Ô∏è Safe Moves: identical files detected by hash; conflicts handled via ‚Äúskip‚Äù or automatic ‚Äú(1) (2) ‚Ä¶‚Äù suffixes  
- üß™ Dry Run: verify everything without moving a single file  
- ‚è™ Undo: restore all moves from the last batch via a journal  
- üß± Filters: ignore clutter (node_modules, .git, __pycache__, hidden/system files as configured)  
- üñ•Ô∏è Cross‚ÄëPlatform UX: open/reveal files using Explorer (Win), Finder (macOS), or xdg‚Äëopen (Linux)

## üöÄ Run from Source
Requirements:
- Python 3.10+
- Tkinter available (Linux: `sudo apt-get install python3-tk`)
- Linux file opening: `sudo apt-get install xdg-utils`

Windows (PowerShell):
```

.\.venv\Scripts\Activate.ps1

```
macOS / Linux:

```

source .venv/bin/activate

```

## üß≠ Usage
1. Home ‚Üí ‚ÄúScan / Preview‚Äù to analyze files.  
2. Watch live actions (MOVE/SKIP/CONFLICT).  
3. Settings:
   - Add Include Paths (e.g., Downloads)
   - Optional Destination Root per include path
   - Exclude globs and conflict policy (skip/suffix)
4. Rules:
   - Map `.ext` ‚Üí `Folder` (e.g., `.jpg` ‚Üí `Pictures`)
   - Falls back to category defaults
5. Sort:
   - Use ‚ÄúDry Run‚Äù to simulate
   - ‚ÄúSort Now‚Äù to apply moves
6. Undo:
   - ‚ÄúUndo ALL‚Äù reverts last batch from the journal

Pro tips:
- Double‚Äëclick a row to open the file‚Äôs folder  
- Right‚Äëclick a row for open/reveal/copy paths

## üß∞ Build Native Executables (PyInstaller)
Build on each target OS for best compatibility.

Windows:
```
pip install pyinstaller
pyinstaller --noconsole --name FileFlow --add-data "src;src" main.py
# Output: dist\FileFlow\FileFlow.exe  (ship the entire dist\FileFlow folder)
```

macOS:
```
pip install pyinstaller
pyinstaller --windowed --name FileFlow --add-data "src:src" main.py
# Output: dist/FileFlow.app  (first run: Ctrl-click ‚Üí Open)
```

Linux:
```
pip install pyinstaller
pyinstaller --windowed --name fileflow --add-data "src:src" main.py
# Output: dist/fileflow  (ship the entire folder)
```

Notes:
- Data separator: ‚Äú;‚Äù on Windows, ‚Äú:‚Äù on macOS/Linux  
- One‚Äëfile build (slower start, self‚Äëextracts): add `--onefile` if desired

## ‚öôÔ∏è Configuration Defaults
- Include Paths: user Downloads by default  
- Exclude Globs: common temp/dev clutter (`.git`, `node_modules`, `__pycache__`, `.DS_Store`, etc.)  
- Conflict Policy:
  - `skip`: do nothing if name exists (identical content SKIPped)
  - `suffix`: make ‚Äúname (1).ext‚Äù, ‚Äúname (2).ext‚Äù, ‚Ä¶
- Categories: Images, Docs, Videos, Audio, Archives, Code, Other

---

## üîí Safety & Undo
- Hash‚Äëbased duplicate detection prevents redundant copies  
- Moves are journaled to `.fileflow_journal.jsonl` under the first include path  
- ‚ÄúUndo ALL‚Äù replays last batch in reverse to restore files

- Open/reveal not working:
  - macOS: `open` is standard
  - Linux: install `xdg-utils` (`xdg-open`)
